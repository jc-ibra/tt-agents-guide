# Anexo A. Guía rápida IDS

**Manual de Uso de GLPI para IDS (Ingenieros de Servicio)**
Trantor Technologies | Service Desk

---

Checklist de una página para consultar desde tu celular en campo.

### Antes de salir

- [ ] Revisaste el ticket completo, no solo el título.
- [ ] Confirmaste dirección y usuario de contacto en las tabs del ticket.
- [ ] Identificaste el equipo y el tipo de servicio a atender.
- [ ] Aplicaste **IDS En camino** con tu hora estimada de llegada.

### Al llegar a sitio

- [ ] Aplicaste **IDS En sitio** con tu hora de llegada.
- [ ] Te presentaste con el usuario de contacto correcto.

### Durante la atención

- [ ] Documentaste el diagnóstico de forma específica (**Diagnosticado**), no genérica.
- [ ] Si dependes de algo externo, aplicaste **Pendiente por cliente** o **Pendiente por refacción** (nunca de forma manual o solo verbal).

### Al resolver

- [ ] Aplicaste la plantilla de **Resolución** correspondiente (Sin refacción / Con refacción / Remota).
- [ ] Recabaste el formato físico de conformidad que aplica para ese cliente.
- [ ] Digitalizaste y adjuntaste el formato al ticket.
- [ ] Si aplica evidencia fotográfica, adjuntaste solo 1 o 2 fotos, no más.
- [ ] Enviaste el formato a tu coordinador, además de adjuntarlo al ticket.

### Reglas de oro

- Si no está en el ticket, no sucedió.
- Documenta en tiempo real, nunca todo junto al final.
- Usa siempre la plantilla correspondiente; el texto libre es la excepción.
- Un ticket no está completo sin el formato de conformidad adjunto en GLPI.
